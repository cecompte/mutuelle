Ceci est un fichier de calcul pour la repartition des contributions de la mutuelle, sur la base des regles suivantes:
- Contributions : 5% du revenu net + 0,5% des economies
- Repartition : uniquement pour les membres ayant moins de 2000e d'economies, de sorte a obtenir un minimum de revenu net pour tous les membres. Ex si ce min est de 150 euros, une personne qui a 0 de revenu net recevra 150e, une qui a 50e de revenu net recevra 100e.


/!\ IMPORTANT : C'est un fichier qui ne fonctionne que sur LibreOffice et qui utilise des macros (des bouts de code en gros), donc il faut les activer à l'ouverture (normalement yaura une fenêtre qui s'affichera pour vous
proposer de le faire). C'est tres frequent que par defaut et pour des raisons de securite, LibreOffice desactive les macros. Si on ne vous propose pas de les activer a l'ouverture du fichier, aller dans les parametres de securite pour les activer.
